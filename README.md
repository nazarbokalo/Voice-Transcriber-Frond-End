# Voice-Transcriber Frontend

**Voice-Transcriber Frontend** is the client-side application for the Voice Transcriber service.  
It provides a user-friendly interface for recording or uploading audio, viewing transcriptions, and managing user accounts.

It connects to the [Voice-Transcriber Backend](https://github.com/nazarbokalo/Voice-Transcriber).

---

## Features

- Interface for uploading audio files.
- View processing status of transcriptions.
- Login and Registration forms.
- View history of transcribed notes.

---

## Tech Stack

- **React.js**
- **JavaScript / TypeScript**
- **Axios** (for API requests)
- **CSS**
- **NPM**

---

## Run Locally

### 1. Clone the repository

Open your terminal and run:
```bash
git clone [https://github.com/nazarbokalo/Voice-Transcriber-Frond-End.git](https://github.com/nazarbokalo/Voice-Transcriber-Frond-End.git)
cd Voice-Transcriber-Frond-End
```

### 2. Install dependencies

```bash
npm install
```
### 3. Run the application
```bash
npm start
```
The Frontend will start at: http://localhost:3000
